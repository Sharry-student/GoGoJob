# 代码步骤说明（面向初学者）

## clean_recruitment_data.py
1. 读取原始 CSV（自动尝试 utf-8-sig / gbk / utf-8）。
2. 修正字段名、去重、解析薪资为整数月薪。
3. 推断省份/城市/区域并统一格式。
4. 修复公司规模与公司类别错位问题。
5. 对职位名称做标准化，提取福利关键词。
6. 对岗位描述拆分为：工作内容、岗位技能要求、岗位福利。
7. 输出 `all_jobs_cleaned.csv` 与清洗统计报表。

## app/services/data_loader.py
1. 读取清洗后的 CSV。
2. 清空并重建 jobs 数据内容。
3. 将行数据映射到数据库字段。
4. 对公司规模做兜底校验，保证格式有效。

## app/services/analytics.py
1. 构建看板统计数据（学历、经验、规模、城市分布）。
2. 构建薪资热力图数据（省级均值与Top10）。
3. 训练预测模型并缓存，减少重复计算。
4. 预测逻辑优先使用同条件样本统计，样本不足再走模型。
5. 匹配逻辑按岗位类别约束并综合多维评分。

## app/routes/main.py
1. 提供页面路由（首页/热力图/预测/匹配/详情）。
2. 提供API路由（统计、预测、匹配、地图明细、模型指标）。
3. 对输入参数做完整性校验，防止前端误请求。
4. 分类页通过 `/api/dashboard?category=...` 获取该类别 Top10 岗位展示列表。

## app/routes/auth.py
1. 处理登录/登出。
2. 管理员新增、删除用户。
3. 初始化默认管理员账户。

## app/__init__.py
1. 创建 Flask 应用对象。
2. 初始化数据库与登录系统。
3. 注册路由蓝图。
4. 提供 `/bootstrap` 执行 CSV -> SQL 同步，并刷新模型缓存。

## UI 关键实现
1. `base.html`：侧栏分族群导航 + 右上角头像菜单（系统管理与登出）。
2. `category.html`：学历/经验图表下方增加 Top10 岗位列表。
3. `predict.html` 与 `match.html`：输入区采用悬浮面板，滚动时保持可操作。
4. `main.css`：深色侧栏、浅色内容区、图表标题防换行、登录背景图遮罩可读性优化。
5. 顶部浅色背景块 `content-head-bg` 内统一承载“管理员”标签、头像与菜单。
6. 管理员头像菜单采用悬停延迟关闭机制，提升菜单交互容错性。
7. 热力图页面左右主容器高度对齐，避免视觉失衡。
